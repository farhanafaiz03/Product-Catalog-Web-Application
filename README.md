# Product Catalog Web Application

## Overview
A modern web-based product catalog system that allows users to browse, search, and manage products with category organization. The application features user authentication, XML data integration, and a responsive interface for an enhanced shopping experience.

## Features
- **User Authentication**: Secure login and registration system
- **Product Catalog**: Browse products with detailed information and images
- **Category Management**: Organize products by categories
- **Product Details**: View comprehensive product information
- **XML Feed Support**: Import and manage product data from external XML feeds
- **Download Functionality**: Export product catalog data in XML format
- **Contact Management**: Get in touch with support through the contact page
- **About Page**: Learn more about the application and company
- **Responsive Design**: Mobile-friendly interface using modern CSS

## Getting Started
1. Open `index.html` in your web browser to start using the application
2. Register a new account or login with existing credentials
3. Browse the product catalog by category
4. View detailed product information
5. Export catalog data as XML format

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- XML for data management

- **auth.js**: Handles user authentication and session management

- **style.css**: Complete styling for responsive design

## Usage
- Navigate through categories to find products
- Use the search functionality to find specific items
- Download the product catalog in XML format
- Manage your user profile through the dashboard
